# ENUM

Esta classe possui o propósito de facilitar a identificação do status de um ``Equipamento``.

```
public enum STATUS_EQUIPAMENTO {
    DISPONIVEL,
    SEM_ESTOQUE,
    DANIFICADO,
    EM_MANUTENCAO,
    ALOCADO
}
```
